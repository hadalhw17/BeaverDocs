enum class EShouldRun : uint8¶

Return type for a function that will be treated as ConditionDesc.

Values:

enumerator No¶

Do not run this frame.

enumerator Yes¶

Run this frame once.

enumerator NoAndCheckAgain¶

Do not run the system, but evaluate the condition at the end of the stage and rerun if positive result is returned.

enumerator YesAndCheckAgain¶

Run the system and test again at the end ofthe stage and rerun if positive result is returned.
