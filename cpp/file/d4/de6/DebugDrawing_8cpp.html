namespace Bvr
namespace DebugDraw

Component debug drawing API.

A condition based API which allows registering a condition per component type To conditionally run debug systems in the Bvr::CSchedule. Registered conditions are visualised in the editor as a checkbox. Advised to use in conjunction with Bvr::Gizmos to draw debug primitives. By default, engine provides draw systems for SMeshComponent, SPointLightComponent and SSpotLightComponent.

Usage:
app
    .AddSystem(SystemDesc(DrawSomeComp)
    .WithCondition(Bvr::DebugDraw::CondFor<T>));
     // Alternatively
    .AddSystem(Bvr::DebugDraw::SystemDesc<T>(DrawSomeComp));
