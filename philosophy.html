Engine philosophy¶

The engine has 3 main execution scopes:

  1. Reflection scope.

  2. Build scope.

  3. Scheduled scope.

Reflection scope¶

  • Happening inside of RAVEN_REFLECTION_BLOCK macro

  • Runs once before main

Build scope¶

  • Where Raven::App initialisation is happening

  • Runs once inside main

Scheduled scope¶

  • Happens after Build Scope is done

  • Runs either:

    • Once if Raven::App::RunOnce() is called

    • Untill app is terminated if Raven::App::Run() is called

Initialisation¶

Code engine initialisation is happening inside of the Build Scope, where all plugins are added to the main application. A plugin must satisfy Raven::Plugin concept.
